# Reymon Database
Es un gestor de bases de datos bastante primitivo.
Solo es un mini proyecto para pasar una materia.

# Dependencias
- `libz` instalada a nivel sistema
- IBM COBOL for Linux 1.2.0 o superior (disponible gratis en ibm.com)

# Compilación
Ejecuta `make` o `make -j$(nproc)` para compilar.

# Ejecución
Este proyecto fue compilado con IBM COBOL for Linux x86.
No se ofrece soporte para GnuCOBOL hasta que el proyecto
evolucione lo suficiente para garantizar paridad de comportamiento.
