# Reymon Database

Es un gestor de bases de datos bastante primitivo.

Solo es un mini proyecto para pasar una materia.

# Dependencias

Debes de poner todas las versiones estáticas de todas las librerías
en en los siguientes directorios:

`lib/berkly`   -> Poner aquí *`libdb-5.3.a`*

`lib/cjson`    -> Poner aquí *`libjson.so`* y *`libcjson.so.1`*

`lib/cobol`    -> Poner aquí *`libcob.a`*

`lib/gmp`      -> Poner aquí *`libgmp.a`*

`lib/libc`     -> Poner aquí *`libc.so`* y *`libc.so.6`* aunque puedes omitirlo en este caso debido a que es una librería estándar del sistema

`lib/libm`     -> Poner aquí *`libm.so`* y *`libm.so.6`* aunque puedes omitirlo en este caso debido a que es una librería estándar del sistema


`lib/libz`     -> Poner aquí *`libz.a`*

`lib/lzma`     -> Poner aquí *`liblzma.a`*

`lib/ncursesw` -> Poner aquí *`libncursesw.a`*

`lib/panelw`   -> Poner aquí *`libpanelw.a`*

`lib/tinfo`    -> Poner aquí *`libtinfo.a`*

`lib/xml2`     -> Poner aquí *`libxml2.a`*

# Compilación

Solo ejecuta `make` o `make -j$(nproc)` para compilar
